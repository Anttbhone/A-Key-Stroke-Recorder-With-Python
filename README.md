# A-Key-Stroke-Recorder-With-Python
This is a key stroke recorder using python. When the script is running every letter on your keyboard that is typed goes into the .txt file.

Safety
_________________
Firstly, this is only for educational purposes only. Do not use it on other people computers or devices. This has no alternatives or relitations to keyloggers.

How to set up
__________________
1. First download all the main.py and the dataKey.txt file.
2. Run the main.py and test the code by adding any type of sentence or a mixture of strings to test.
3. And after you are done you can check in the dataKey.txt file of what you wrote.
